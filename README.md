# Claude Usage Tracker - Update Feed

This branch hosts the Sparkle appcast.xml file for automatic updates.

- **Appcast URL**: https://hamed-elfayome.github.io/Claude-Usage-Tracker/appcast.xml
- **Generated by**: GitHub Actions workflow (generate-appcast.yml)
- **Do not edit manually**: This file is automatically generated on each release

## How it works

1. When you tag a release (e.g., v2.0.0), the release workflow builds and publishes
2. The generate-appcast workflow automatically updates appcast.xml
3. Users running the app get notified of the update within 24 hours
